# Olist E-Commerce Sales Dashboard

## Overview
Built an interactive Power BI dashboard using the Olist Brazilian E-Commerce dataset to analyze sales performance, customer behavior, and product trends.

## KPIs
- Total Revenue
- Total Orders
- Total Customers
- Average Review Score

## Visualizations
- Top 5 Product Categories by Revenue
- Monthly Order Trend
- Revenue by Customer State
- Order Status Distribution

## Features
- Interactive Year Slicer
- Dynamic KPI Cards
- Cross-filtering across visuals

## Tools Used
- Power BI
- SQL Server
- DAX
